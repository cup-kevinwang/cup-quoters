Pricing Notes
  • Packing: Base on yellow carton. White carton: +US$0.10/ctn
  • Container: Base on 40HQ. If 20GP: 4oz +$0.20 / 8oz +$0.40 / 10-22oz +$0.50 /ctn
  • Printing: Base on 1-3 colors. Extra color: 4-8oz +$0.10 / 10-22oz +$0.20 /ctn
  • Innercup GSM: ±10 gsm per step. Inner liner paper. Affects base paper cost by ratio.
  • Print GSM: ±1 gsm per step. Print coating layer. Affects only printing cup cost.
  • Kraft GSM: ±1 gsm per step. Kraft ripple layer. Affects both cup costs by ratio.
  • Fan/Bottom price: RMB per ton. Affects paper cost directly.
  • All Items: 500 pcs / ctn
Formula
  • Base paper (Innercup+Bottom) = cp0 × (Fan×IcGSM×IcArea + Bottom×BgGSM×BtArea) / origBaseMetric
  • Kraft layer cost = kfCost0 × (Fan×newKfGSM) / (6400×origKfGSM)
  • Print layer cost = prCost0 × (Fan×newPrGSM) / (6400×origPrGSM)
  • Paper(Kraft) = Base + Kraft layer  |  Paper(Print) = Base + Kraft + Print layers
  • Cost = Paper + Cut + Form-Pack + Depre + PumpE/Qty
  • Price = Cost / ExRate + Margin / (Qty × ExRate)
