Pricing Notes
  • Packing: Base price on yellow carton. White carton: +US$0.10/ctn
  • Container: Base price on 40HQ. If 20GP: 4oz +$0.20 / 5-7oz +$0.30 / 8oz +$0.40 / 9-32oz +$0.50 /ctn
  • Printing: Base on 1–3 colors. Each extra color: 3–9oz +$0.10 / 10–32oz +$0.20 /ctn
  • Fan GSM: Each ±10 gsm adjusts paper cost by ratio. Modify directly in the GSM row on the Quote page.
  • Fan/Bottom price: RMB per ton. Affects paper cost directly.
Formula
  • Paper cost = (fan_area × fan_gsm × fan_price + bottom_area × bottom_gsm × bottom_price) / 10¹² × pcs/ctn
  • Cost/Print ctn = Paper + Print-Cut + Form-Pack-Trans + Depre + PumpE/qty_40HC
  • Cost/White ctn = Paper + Form-Pack-Trans + Depre + PumpE/qty_40HC + Gap
  • Price = Cost / ExRate + Margin / (qty_40HC × ExRate)
